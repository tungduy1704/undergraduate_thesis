## Overview

This repository contains the input files, scripts, and computed data for my undergraduate thesis, which investigates the **optical matrix elements** of monolayer **Transition Metal Dichalcogenides (TMDs)** — a class of two-dimensional materials with exceptional optoelectronic properties — using a three-bands tight-binding model parameterized from first-principles calculations.

The central goal is to analytically and numerically derive the optical matrix elements that govern light–matter interactions in TMD monolayers (MoS₂, WS₂, MoSe₂, WSe₂), enabling predictions of optical selection rules, valley-dependent responses, and Berry curvature effects.
